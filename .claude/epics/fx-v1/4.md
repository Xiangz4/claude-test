# 开发商户端界面

## Description

开发商户端换汇下单页和订单管理页，实现WebSocket实时汇率推送和订单状态更新，复用Shoplazza设计系统组件

## Acceptance Criteria

- [ ] 换汇下单页实现（币种选择、金额输入、汇率展示、30秒倒计时）
- [ ] 支持买入/卖出金额双向计算
- [ ] 汇率锁定和下单流程实现
- [ ] 2FA安全验证集成
- [ ] 订单列表页实现（筛选、分页、状态标签）
- [ ] 订单详情抽屉实现（完整信息展示）
- [ ] WebSocket实时推送汇率更新
- [ ] WebSocket实时推送订单状态变更
- [ ] 订单导出功能（Excel）
- [ ] 移动端响应式适配
- [ ] 中英双语切换

## Technical Details

- 使用React + TypeScript + React Query
- 使用React Hook Form + Zod表单校验
- 使用WebSocket订阅汇率和订单事件
- 倒计时组件使用useInterval Hook
- 复用Shoplazza UI组件库
- Mock数据提前开发，后期对接真实API

## Notes

Can run in parallel with backend tasks using mock data
