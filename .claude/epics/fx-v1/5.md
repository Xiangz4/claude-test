# 开发运营端界面

## Description

开发运营端汇率配置管理页、商户配置页、订单管理页、工单审批流程，实现数据统计看板

## Acceptance Criteria

- [ ] 通道汇率查看页实现（表格展示历史汇率）
- [ ] 平台标价汇率配置页实现（表单+预览+一键配置）
- [ ] 商户定价汇率配置页实现（商户选择+批量配置+单独议价）
- [ ] 商户权限管理页实现（配置可用币种对）
- [ ] 订单管理页实现（高级筛选+多Tab展示+导出）
- [ ] 订单详情页实现（商户信息+订单详情+通道信息+收益信息）
- [ ] 工单审批流程集成（特殊汇率审批+异常处理）
- [ ] 数据统计看板实现（GMV、成功率、收益等）
- [ ] 配置变更历史记录展示

## Technical Details

- 使用React + TypeScript
- 使用TanStack Table实现复杂表格
- 使用ECharts实现数据图表
- 配置预览功能：实时计算加点后的汇率
- 工单审批对接运营管理系统API
- 数据看板定时刷新
